Summary: Visual Studio Code Agents の初見レビュー

Visual Studio Code Insiders の2026年4月の更新で、Visual Studio Code Agents という機能が追加されました。

https://code.visualstudio.com/updates/v1_115

リリースノートの説明によると、Visual Studio Code Agents は、新しいプレビュー用のコンパニオンアプリであり、エージェントネイティブ開発向けに構築されているようです。

Visual Studio Code Agents (長いので、VSC Agents と略します) は、VSCode (IDE) とは別のアプリケーションとして動作します。追加インストールは不要で、VSCode Insiders をインストールしていれば利用可能です。
