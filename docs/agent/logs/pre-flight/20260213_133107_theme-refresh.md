#### Preflight
* 目的: Gemini提案のデザインを既存Pelicanテーマへ反映し、ページ構成を維持したまま見た目のみ刷新する。
* 変更範囲: `themes/shima-island/templates/` と `themes/shima-island/static/css/` を中心に変更し、必要に応じてテンプレート内のインラインスタイルを調整する。`content/` やPelican設定ファイルは変更しない。
* 前提: SITEURLや公開URLは変更しない。言語は日本語、タイムゾーンは既存設定（Asia/Tokyo）を維持する。
* 手順:
  1. 既存テーマのテンプレート構成とCSS適用箇所を確認する。
  2. 指定デザインをベースに共通スタイルをCSSへ実装する。
  3. index/article系テンプレートのマークアップクラスを新デザインに合わせて調整する（構成は維持）。
  4. `uv run ruff check .` と Pelicanビルドを実行して検証する。
  5. 差分確認後、post-flightログ作成、コミット、PRメッセージ作成を行う。
* 検証コマンド: `uv run ruff check .`、`uv run pelican content -o output -s publishconf.py`
* 相談が必要な点: 依存追加・SITEURL変更・大規模ディレクトリ変更は行わないため事前相談事項なし。
