#### Preflight
* 目的: `index.html` と `article.html` にSNS連携用のmeta（Twitterカード/OGP相当）を追加し、記事ページでは記事情報、一覧ページではサイト情報を出し分ける。
* 変更範囲: `themes/shima-island/templates/` 配下のテンプレート、`docs/agent/logs/` 配下の作業ログ。`content/` や Pelican 設定ファイルは変更しない。
* 前提: 公開URLは既存設定（`SITEURL`）を使用し、記事には `title` / `summary` / 必要に応じてサムネイル画像情報が与えられる想定。Twitterカードは `summary` を利用する。
* 手順: 
  1. 現在のテンプレート構成（`base.html` / `index.html` / `article.html`）を確認する。
  2. `index.html` にサイトタイトル・概要向けのmetaを追加する。
  3. `article.html` に記事タイトル・概要・サムネイル画像向けのmetaを追加する。
  4. Pelicanビルドを実行して生成が成功することを確認する。
  5. Postflightログに変更点と検証結果を記録する。
* 検証コマンド: 
  * `uv run pelican content -o output -s pelicanconf.py`
* 相談が必要な点: なし（依存関係追加、SITEURL変更、大規模構造変更は実施しない）。
