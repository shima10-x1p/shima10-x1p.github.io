Pico.css v2で構築する、
ミニマルでポップなブログデザイン

最近、複雑なCSSフレームワークに疲れてしまい、原点回帰としてPico.cssを触っています。 クラス名をほとんど書かずにセマンティックなHTMLだけで完結する思想は、書くことに集中したいブログという媒体に最適だと感じました。

なぜPico.cssなのか

Tailwind CSSは素晴らしいですが、個人のミニブログには少しオーバーキルな場合があります。 Pico.css v2では、CSS変数の活用によりカスタマイズ性が大幅に向上しており、ダークモードへの対応も一瞬で完了します。

画面右側のサイドバーを撤去し、記事本文に集中できるシングルカラムのデザインに変更しました。これにより、視線の移動が減り、没入感のある読書体験を提供できます。

コードブロックの例

Pythonでの設定ファイルは以下のようになります。

AUTHOR = 'Takumi'
SITENAME = "Takumi's Log"
SITEURL = ""

PATH = 'content'
TIMEZONE = 'Asia/Tokyo'
DEFAULT_LANG = 'ja'
